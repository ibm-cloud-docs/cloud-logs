# DataPrime UUID functions
{: #dataprime-ref-uuid-func}

This guide provides a full glossary of all available DataPrime UUID functions.
{: shortdesc}


## `isUuid`
{: #isuuid}

Returns true if UUID is valid.

```text
isUuid(uuid: string): bool
```
{: codeblock}



## `randomUuid`
{: #randomuuid}

Returns a random UUIDv4.

```text
randomUuid(): string
```
{: codeblock}
